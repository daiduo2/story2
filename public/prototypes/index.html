复古 UI 模板集

视觉风格讨论稿 · 不代表最终上线版本

本目录用于集中展示「白鹿疗养院」不同内容类型的复古档案 UI 模板。每个模板对应一类内容页面的视觉与信息架构，供团队讨论确定最终风格后，再迁移到生产样式表。

  • 01. 元信息页模板（meta）

    适用于首页、关于本项目、访客须知、档案目录等框架性页面。强调整体机构标识与导航结构。

  • 02. 正编病历模板（volume）

    适用于卷一至卷二十四等主体病历文档。突出卷号、年代、科室、页码等档案元信息。

  • 03. 补遗档案模板（supplement）

    适用于林素琴日志等个人补遗材料。强调作者身份、手写痕迹与主观叙事。

  • 04. 外围档案模板（peripheral）

    适用于外部机构提供的监控、药房、供电等记录。强调来源机构与证据编号。

  • 05. 变体页模板（variant）

    适用于觉醒版、异常状态页面。展示反色、扫描线、系统错误弹窗等异常视觉。
